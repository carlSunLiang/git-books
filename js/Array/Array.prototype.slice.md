**slice()** 返回当前数组的指定部分的副本到一个新数组。

### 语法

```
arr.slice(begin[, end]);
```

### 参数

**begin**

提取数据的开始下标(数组下标以0为开始)，参数值可以为负，可以不指定

```
var arr = [1, 2, 3, 4, 5],
    arrCopy;

// 常规使用， 从第三个元素开始截取
arrCopy = arr.slice(2);   // [3, 4, 5]

// 负值，从倒数第二个开始截取
arrCopy = arr.slice(-2);   // [4, 5]

// 不填，复制所有元素
arrCopy = arr.slice();     // [1, 2, 3, 4, 5]
```
**end**

提取数组的结束下标，不包含结束下标的元素
